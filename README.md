# ARKDEMIA — Road to Battle Coliseum

Bem-vindo ao repositório oficial do **ARKDEMIA**, um projeto dedicado à comunidade de **Fighting Games (FGC) do Amapá**.

Este projeto apresenta o ranking, resultados e informações sobre os torneios e eventos realizados, com foco em jogos como **Street Fighter 6** e **2XKO**.

## Estrutura do Projeto

O projeto foi organizado para facilitar a manutenção e o desenvolvimento:

- **`assets/images/`**: Contém todas as imagens, logos e ícones utilizados no site.
- **`css/`**: Contém os arquivos de estilo (`style.css`), responsáveis pelo design e identidade visual.
- **`js/`**: Contém os scripts JavaScript (`app.js`) que gerenciam a lógica do ranking e interatividade.
- **`index.html`**: Página principal do projeto.
